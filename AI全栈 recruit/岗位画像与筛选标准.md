## 岗位基本信息

- 岗位名称：AI 全栈工程师
- 地点：北京
- 薪资范围：30-50K（实际给薪看人定级，通过初筛者中有人要 20-25K）
- 方向：AI Agent 工程化落地 + 全栈开发

## 岗位核心需求

从通过初筛的 4 份简历反推，这个岗位的核心需求不是传统"前端+后端"，而是 **AI Agent 工程化落地 + 全栈能力**。

### 硬性门槛

| 维度 | 标准 |
|---|---|
| 学历 | 985/211 本科及以上，计算机/软件/AI 相关专业 |
| AI Agent 项目 | 必须有实际落地的 Agent 项目（不是 Demo/课程作业） |
| RAG 知识库 | 亲自做过文档向量化、检索增强生成，知道 chunk size / TopK 调优 |
| 全栈能力 | 后端（Python/Java）+ 前端（React/TS）都能写 |

### 核心竞争力（通过初筛的关键加分项）

1. Agent 架构设计 — 多智能体协作、图编排（LangGraph/SpringAI 图编排）、ReAct 模式
2. MCP 协议 / Skills 体系 — 工具集成、插件化 AI 框架
3. RAG 全链路 — 不做 API 调包，做过文档分割策略、混合检索（BM25+向量）、Reranker 重排序，有准确率数字
4. 工程化能力 — SSE 流式输出、状态机、权限控制、可观测性、E2E 测试

### 加分但不必须

- 大厂经验（华为、百度等）
- 硕士学历
- 算法背景 / 数学建模获奖

## 筛选判断规则

### 直接淘汰
- 无 AI Agent / RAG 实际项目经验（仅 API 调用/ChatGPT 套壳不算）
- 无全栈能力（只会前端或只会后端）
- 学历明显低于 211

### 可以通过初筛
- 有 1 个以上 AI Agent 真实落地项目 + RAG 系统搭建经验 + 前后端都能写
- 技术栈命中：LangChain/LangGraph/SpringAI + MCP + 向量数据库 + React/TS + Python/Java
