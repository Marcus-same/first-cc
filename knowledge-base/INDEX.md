## 🔍 快速检索

- **全文搜索**：`python kb.py search <关键词>`
- **飞书搜索**：`lark-cli base +record-search --base-token <token> --table-id <table> --keyword <关键词>`
- **本地 grep**：`grep -r "关键词" knowledge-base/ --include="*.md"`
