## 背景
blog_monitor 目前只支持小红书博主监控，需要新增微信公众号文章监控。

## 决策

### 方案选型
- **WeWe-RSS (Docker)**：用微信读书账号订阅公众号，自动生成 RSS feed
- 排除搜狗微信爬虫（反爬严重、不稳定）
- 排除公众号 API（只能管理自己的号）

### 架构
```
WeWe-RSS (Docker) → RSS feeds → blog_monitor → AI摘要 + 飞书推送
                                            → 原文保留（清洗HTML）
```

### 关键需求
- 不仅要 AI 摘要，**必须保留公众号原文全文**
- 接入现有 monitor.py 的轮询/去重/推送/Web面板流程
- Docker Compose 部署，SQLite 模式（轻量）

## 产出
- `blog_monitor/wewe-rss/docker-compose.yml` — WeWe-RSS 容器配置（FEED_MODE=fulltext 保留原文）

## 阻塞
- Windows 需要先安装 WSL2，Docker Desktop 才能启动
- 等待执行 `wsl --install`（需管理员权限+重启）
