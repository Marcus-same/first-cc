# 初中数学讲义-考情匹配分析项目

## 背景
孟需要对比**拾光讲义**与**哈尔滨中考真题**的匹配度，输出两个维度的结论：
1. 拾光讲义中与哈尔滨考情匹配的内容（精确到讲次、小标题）
2. 拾光讲义中未覆盖的哈尔滨考情内容

## 项目位置
`D:\projects\math-curriculum-matching\`

## 核心产出

### 1. 397题型知识库
- 来源：《2025初中数学题型方法全归纳》（陈吉/刘云龙编著）
- 覆盖：代数189 / 几何124 / 函数55 / 统计概率29
- 每题型含：五维评分(kn/kc/td/cl/tp)、前置依赖、原5级难度

### 2. 六级难度体系（融合一组方法论）
学习了一组（重庆）的产出后，从五级升级为六级：
- Lv1 单点结构 → Lv2 双点结构 → Lv3 多点结构 → Lv4 关联结构(分类讨论) → Lv5 抽象扩展(策略规划) → Lv6 形式运算(创造性破题)
- 附加 SOLO 理论映射 + 60道标杆校准题 + 试卷位置锚定

### 3. 对照实验体系
分为两组，同一数据独立运行：

| 维度 | Group A（对照组/学习前） | Group B（实验组/学习后） |
|------|--------------------------|--------------------------|
| 难度 | 五级 lv1-lv5 | 六级 Lv1-Lv6 + SOLO |
| 匹配 | 单一关键词 | 三维（关键词+相似度+难度对齐） |
| 缺口 | 题型级 | 子类型级 + 补全方案 |
| 定位 | 不考虑 | 讲义定位评估 + 严重性校准 |

### 4. 分析引擎
- `extract_exam.py`：真题Word → 逐题提取 → 题型匹配
- `match_syllabus.py`：讲义Word → 章节提取 → 覆盖检测
- `compare.py`：对比报告生成（匹配明细 + 缺口清单 + 补全方案）

## 关键决策
1. **难度从五级升六级**：一组方法论验证了Lv4（分类讨论）和Lv6（创造性破题）需要独立分层
2. **三维匹配替代单一关键词**：减少假阳性，提高缺口诊断精度
3. **子类型拆分**：缺口必须拆到子类型（如"分式增根22道/含参分式28道"），否则补全方案无法排课
4. **讲义定位框架**：评价前先判断讲义是基础型/培优型/冲刺型，Lv1-Lv2缺失在培优型中不算缺陷
5. **对照实验设计**：两组独立pipeline，同数据不同方法，验证升级效果

## 一组（重庆）关键发现（交叉验证）
- 新定义题是最大单一缺口（145题，12.9%）
- 分式方程与不等式参数严重缺失（220题，19.5%）
- 统计概率基础层次缺失（220题选填）
- 二次函数需扩充几何综合
- 讲义定位"中考决胜营"=培优提升型，Lv1-Lv2缺失为设计预期
