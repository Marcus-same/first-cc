## 四、配套提示词工程（Prompt Engineering）

### 4.1 提示词设计思路

将以上 397 种题型的分级体系转化为 LLM 可用的提示词，核心设计原则：

1. **分层标注**：不直接给 AI 看 397 条，而是用规则+少量示例让 AI 学会分级逻辑
2. **Few-shot 策略**：每个等级提供 3-5 个标准示例，让 AI 内化难度边界
3. **条件触发**：根据题型名称、知识范围、步骤数量来判断难度
4. **自检机制**：让 AI 对自己的分级给出理由，方便人工复核

### 4.2 题型提取提示词

```
你是一个专业的初中数学教研专家。请从以下数学教材/教辅文本中，
提取所有"题型方法"，并按5级难度标准进行分级。

## 难度分级标准

### lv1 基础认知
- 涉及1个知识点，概念辨识或一步计算
- 示例："求一个数的相反数""判断必然事件""科学记数法"
- 典型关键词：判断是否、概念、书写规范、性质（单一）

### lv2 基础应用
- 涉及2个相互联系的知识点，需要多步运算或简单分析
- 示例："整式化简求值""解一元一次不等式""求一次函数解析式"
- 典型关键词：化简求值、解法、应用题、图象与性质、判定

### lv3 中等综合
- 涉及多个知识点，需要一定技巧和中等计算
- 示例："含参数的一元一次方程""配方法求最值""反比例函数k的几何意义"
- 典型关键词：含参、换元、模型(基础几何模型)、综合应用、方案设计

### lv4 难题/压轴第一问
- 涉及多个强关联知识点，需要构造辅助线或较强分析能力
- 示例："倍长中线""将军饮马模型""一次函数与将军饮马""区间最值动轴定区间"
- 典型关键词：辅助线构造、动点、手拉手、一线三等角、翻折综合

### lv5 压轴最后一问/竞赛
- 涉及多知识点深度综合，5步以上逻辑链，大量复杂计算或高难辅助线
- 示例："费马点""阿氏圆问题""胡不归问题""托勒密定理""探照灯模型"
- 典型关键词：费马点、阿氏圆、瓜豆原理、托勒密、胡不归、婆罗摩笈多、三动点

## 输出格式
每条题型输出为JSON格式：
{
  "id": 序号,
  "topic": "代数/几何/函数/统计概率",
  "chapter": "章名",
  "type_name": "题型名称",
  "difficulty": "lv1/lv2/lv3/lv4/lv5",
  "reason": "简短分级理由（不超过30字）"
}
```

### 4.3 难度分级自检提示词

```
你刚才对一批初中数学题型进行了难度分级。现在请自检：

## 自检规则
1. lv1中的题型，不应该出现"含参""求参""综合""应用""模型"等词
2. lv5中的题型，必须有"辅助线构造""多知识综合""复杂计算"中的至少两项特征
3. 统计概率类题型最高不应超过lv3
4. 几何中【扩展】标签的题型，难度至少lv3起步

## 请逐条检查并修正
对于每一处修正，说明原始分级、修正后分级、修正理由。
```

### 4.4 批量分级流水线提示词

```
## 任务
对以下{count}条初中数学题型进行批量难度分级。

## 输入格式
每条格式：{章节} | {题型名称} | {原书标签（基础/培优/扩展）}

## 分级逻辑（按优先级依次匹配）

1. **直接命中规则**：题型名称完全匹配以下列表，直接赋予对应等级
   - lv5白名单：费马点、阿氏圆问题、瓜豆原理、胡不归问题、托勒密定理、托勒密不等式、
     西姆松定理、阿基米德折弦定理、探照灯模型/定角定高、婆罗摩笈多、半角模型、三动点问题
   - lv4白名单：倍长中线、截长补短法、手拉手模型、一线三等角、将军饮马模型、隐形圆、
     奔驰模型、十字架模型、区间最值(动轴/动区间)、根的分布、脚拉脚模型、等邻边对角互补

2. **关键词匹配规则**：
   - 含"辅助线""构造""动点""动态"且原标签=培优 → lv4起步
   - 含"含参""求参""换元"且原标签=培优 → lv3起步
   - 含"模型"且非lv4/lv5白名单 → lv3

3. **标签兜底规则**：
   - 原标签=基础 + 不含多步骤关键词 → lv1
   - 原标签=基础 + 含多步骤关键词 → lv2
   - 原标签=培优 + 不匹配lv4/lv5规则 → lv3
   - 原标签=扩展 + 不匹配lv4/lv5规则 → lv3

4. **终验规则**：
   - 统计概率类最高lv3
   - 仅概念辨识/单步运算 → 最高lv1

## 输出格式
每行一条：{id}|{topic}|{chapter}|{type_name}|{difficulty}|{reason}
```
