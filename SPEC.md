# 番茄钟桌面应用 - 规范文档

## 1. 项目概述

- **项目名称**: Pomodoro Timer (番茄钟)
- **项目类型**: 桌面应用 (Electron)
- **核心功能**: 帮助用户使用番茄工作法保持专注，通过计时器、休息循环和统计功能提升工作效率
- **目标用户**: 需要时间管理和专注力提升的用户

## 2. UI/UX 规范

### 2.1 窗口配置
- **窗口尺寸**: 400 x 550 像素（最小尺寸 400 x 500）
- **窗口特性**: 可最小化到系统托盘，点击关闭按钮隐藏到托盘而非退出
- **窗口样式**: 原生窗口边框

### 2.2 布局结构
```
┌─────────────────────────────┐
│     模式标签 + 番茄计数      │  (顶部区域)
├─────────────────────────────┤
│                             │

│                             │
├─────────────────────────────┤
│     ▶ 开始/暂停  ↺ ⏭        │  (控制按钮)

```

## 3. 功能规范

### 3.1 核心功能

#### 番茄计时器
- 默认工作时长: 25 分钟
- 默认短休息: 5 分钟
- 默认长休息: 15 分钟
- 每 N 个番茄后长休息: 默认 4 个
- 计时精度: 秒级

#### 计时控制
- **开始**: 启动倒计时
- **暂停**: 暂停倒计时，保持当前状态
- **重置**: 重置当前模式到初始时间
- **跳过**: 直接切换到下一模式

#### 模式切换
1. 工作 → 短休息 (默认)
2. 工作 → 长休息 (每 N 个番茄后)
3. 休息 → 工作

### 3.2 设置功能
- 工作时长: 15-60 分钟，默认 25
- 短休息: 1-15 分钟，默认 5
- 长休息: 5-30 分钟，默认 15
- 番茄计数阈值: 2-8，默认 4
- 设置自动保存到 localStorage

### 3.3 桌面通知
- 专注结束时: "专注完成！干得不错，休息一下吧~ 🍅"
- 短休息结束: "休息结束！准备好继续专注了吗？"
- 长休息结束: "长休息结束！充满电了，继续加油！💪"

### 3.4 系统托盘
- 点击关闭按钮隐藏到托盘
- 托盘菜单: "显示窗口" / "退出"
- 支持通过托盘图标恢复窗口

### 3.5 每日统计
- 统计项:
  - 今日完成番茄数
  - 今日累计专注时长
  - 今日工作 sessions 数
- 数据存储: localStorage，按日期区分
- 统计重置: 每日 0 点自动重置（或首次使用时检测新日期）

## 4. 技术规范

### 4.1 技术栈
- **框架**: Electron 33.x
- **渲染**: HTML5 + CSS3 + Vanilla JavaScript
- **构建**: electron-builder

### 4.2 文件结构
```
pomodoro-timer/
├── assets/
│   └── icon.png          # 应用图标
├── src/
│   └── main.js            # Electron 主进程
├── index.html             # 渲染进程入口
├── package.json
├── SPEC.md
└── README.md
```

### 4.3 IPC 通信
- `show-notification`: 主进程显示系统通知
- `quit-app`: 主进程退出应用

### 4.4 数据持久化
- 设置: localStorage (`pomodoro-settings`)
- 统计: localStorage (`pomodoro-stats`)
  ```json
  {
    "2026-05-08": {
      "pomodorosCompleted": 4,
      "totalFocusMinutes": 100,
      "sessionsCompleted": 4
    }
  }
  ```
